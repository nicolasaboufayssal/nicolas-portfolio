Crypto Master – A Domain‑Specific Knowledge Engine

Python · LangChain · FAISS

The crypto landscape evolves at breakneck speed, making it difficult to cut through hype and find real insights. To tackle this problem I built Crypto Master, a personalised research assistant that ingests and organises knowledge from dozens of blockchain books. A Python ETL pipeline cleans and normalises each text before generating vector embeddings with FAISS via LangChain. The embeddings are served through a Flask API so that users can ask natural‑language questions and receive context‑aware answers in seconds.

Along the way I grappled with everything from inconsistent OCR to tuning embedding dimensions for semantic search. The project sharpened my data engineering skills and showcased how language models and vector databases can turn static books into an interactive knowledge base. Crypto Master continues to be my go‑to tool for staying on top of blockchain developments and inspires ideas for new domain‑specific assistants.

WhatsApp Bot – Automating Client Support

Node.js · Twilio · MongoDB

As more clients started contacting us through WhatsApp I saw an opportunity to streamline responses. I designed and built a workflow bot that sits between Twilio’s WhatsApp API and our internal systems. Written in Node.js with a MongoDB backend, the bot uses simple natural‑language classification to understand intent, dispatches personalised replies and escalates complex queries to the right human via our ticketing system.

By handling hundreds of messages per day automatically it eliminated repetitive follow‑ups and improved response times. Building it taught me how to design conversational flows, integrate messaging platforms with business processes and balance automation with the need for human empathy.

Social Media Bot – Content Scheduling on Autopilot

Python · Selenium · Cron

Maintaining a consistent social‑media presence is essential yet time‑consuming. I built a headless Python script using Selenium that logs into TikTok, Instagram and YouTube, uploads media and captions and schedules posts via cron. The bot abstracts away the web interface and runs nightly, queueing up posts for the week ahead without any manual intervention.

This automation freed up hours of creative time and taught me the nuances of automating web interactions responsibly—dealing with CAPTCHAs, rotating user agents and respecting platform limits. It demonstrates how a few hundred lines of code can reclaim dozens of hours of repetitive work each month.

Custom GPT Agents – Tailored AI Assistants

OpenAI API · Express

Intrigued by the rise of generative AI, I decided to build my own assistants. Using the OpenAI API and an Express backend I designed a suite of custom GPT agents that connect to our internal ticketing and CI/CD dashboards. Each agent chains prompts together with real‑time context to answer support questions, generate research summaries, draft release notes or help onboard new team members.

I deployed five such assistants covering domains from DevOps to HR. They automate repetitive knowledge work and prove that AI can be woven into everyday workflows. Over time they’ve become an indispensable part of my toolkit—accelerating tasks, amplifying creativity and inspiring others to imagine how AI can augment their own roles.
